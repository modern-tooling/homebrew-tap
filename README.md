# modern-tooling Homebrew Tap

Homebrew formulas for [modern-tooling](https://github.com/modern-tooling) projects.

## Installation

```bash
brew tap modern-tooling/tap
brew install work-lab
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| [work-lab](https://github.com/modern-tooling/work-lab) | Container-based lab for humans and AI coding agents |

## Updating

```bash
brew update
brew upgrade work-lab
```
